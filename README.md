# Email Project

## Project Description
This project is designed to help users manage their emails efficiently by providing features for sorting, searching, and organizing messages.

## Features
- **Email Sorting**: Automatically sort emails by date, sender, or subject.
- **Search Functionality**: Quick search options to find specific emails.
- **Labeling System**: Users can label emails for easier organization.
- **Spam Detection**: Built-in spam detection to keep inboxes clean.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Heroku

## Usage Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Amincsed26/Email.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Email
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to access the application.
